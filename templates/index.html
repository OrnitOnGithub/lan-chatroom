Information

Hello and welcome to the chatroom.

We are currently even before the alpha release so you might encounter a few issues using this chatroom.
A current known issue is a race condition between the network library and the flask web app, sometimes resulting in spontaneously clearing the chat.
The networking library is quite a mess and may cause some problems of its own.

Also, as was mentioned in the README, you cannot see chats from before you joined the chatroom.

If you want to report issues, please do so here here.

Thank you.
